# Installation

## Package Manager

Install Casque using your preferred package manager:

```bash
# npm
npm install casque

# yarn
yarn add casque

# pnpm
pnpm add casque

# bun
bun add casque
```

## Usage

Import the functions you need:

```typescript
import { element, row, column } from "casque";
```

## TypeScript

Casque is written in TypeScript and includes type definitions out of the box. No additional `@types` package is needed.

```typescript
import type { Element, Component, Anchor, Margin } from "casque";
```
